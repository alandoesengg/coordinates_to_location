# Location Mapping Using Latitude and Longitude

## Description
This project maps a geographical location using user-provided latitude and longitude values. 
The map updates dynamically and displays a marker at the specified coordinates.

## Technologies Used
- HTML
- CSS
- JavaScript
- Leaflet.js
- OpenStreetMap

## Working Principle
1. User enters latitude and longitude values.
2. Inputs are validated for valid coordinate ranges.
3. Leaflet map is centered on the entered coordinates.
4. Marker position is updated dynamically.

## Validation Rules
- Latitude must be between -90 and 90.
- Longitude must be between -180 and 180.

## Output
An interactive map with a marker pointing to the entered location.
